# Discord Channel Message Cloner

A powerful Discord bot that clones messages from a channel in one server to a channel in another server in real time.

The bot automatically reads messages from a source channel and reposts them to a destination channel, making it ideal for mirroring announcements, syncing community updates, or creating backups across multiple Discord servers.

## Features

* Real-time message cloning between Discord servers
* Supports text messages, embeds, and attachments
* Easy configuration of source and destination channels
* Lightweight and efficient operation
* Permission validation to ensure proper access

## Requirements

The bot must have the following permissions:

### Source Server

* View Channel
* Read Message History

### Destination Server

* View Channel
* Send Messages
* Embed Links (if cloning embeds)
* Attach Files (if cloning attachments)

## Use Cases

* Cross-server announcement synchronization
* Community network management
* Backup and archival channels
* Multi-server content distribution

Simply invite the bot to both servers, grant the required permissions, and start mirroring messages instantly.
